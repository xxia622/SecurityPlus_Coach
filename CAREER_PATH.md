# Cybersecurity Career Path — Personal Roadmap
**UCR CSBA → Red Team / Penetration Testing / Web3 Security**

---

## 整体方向定位

CSBA 学位（CS + 商业）在安全行业是个优势——技术+业务双轨，特别适合**安全咨询**和**Web3审计**这两个高薪方向。

---

## 阶段一：在校期间（现在 → 2027年秋）

**目标：建立基础 + 拿到第一个实习**

| 时间 | 行动 |
|------|------|
| 2026年6月 | 通过 **Security+** |
| 2026年夏 | 在 TryHackMe 刷完 Jr Penetration Tester 路径 |
| 2026年秋 | 拿 **eJPT**（入门级实战渗透证书，$200，含实验） |
| 2026年秋冬 | 打 CTF 比赛，建 GitHub 项目组合 |
| 2027年夏 | **网络安全实习**（目标：渗透测试 / AppSec 方向） |

**现在就要练的技能：**
- Linux 命令行 + bash scripting
- Python（写漏洞利用脚本）
- 网络基础（Wireshark、TCP/IP、子网）
- OWASP Top 10（Web漏洞基础）
- Burp Suite

---

## 阶段二：毕业后 Entry Level（2027 – 2029）

**理想起点职位（按优先级）：**

| 职位 | 为什么适合你 | 薪资范围 |
|------|------------|---------|
| **Junior Penetration Tester** | 直接对口，小型安全咨询公司容易进 | $75k–$100k |
| **SOC Analyst T2** | 学蓝队思维，让你以后攻击更精准 | $65k–$90k |
| **Application Security Engineer** | CS背景优势，代码审计 + 漏洞挖掘 | $90k–$120k |

> **建议：** 如果能直接进渗透测试岗就进，否则先做 SOC T2 积累1年再跳。不建议从 T1 起步，浪费时间。

**这阶段必拿的证书：**
- **OSCP**（Offensive Security Certified Professional）— 渗透测试行业黄金标准，没有它进不了顶级红队，目标2028-2029年拿

---

## 阶段三：Mid Level（2029 – 2031）

**目标职位：**
- Penetration Tester（独立渗透测试）
- Red Team Analyst
- Smart Contract Security Auditor（如果走Web3方向）

**证书：**
- **OSEP**（高级红队、C2框架、免杀）
- **CRTO**（Cobalt Strike红队操作，实战性极强）
- Web3方向：无通用证书，靠**审计报告组合**说话

---

## 阶段四：Senior Level（2031+）

**目标职位：**
- Senior Penetration Tester / Red Team Lead — $150k–$200k+
- Security Consultant（咨询公司：Mandiant、CrowdStrike、NCC Group）
- Smart Contract Auditor — 顶级公司：Trail of Bits、Halborn、Certik

## Web3 安全专项路径（差异化优势）

这个方向目前竞争少、薪资高，值得专项投入：

1. 学 **Solidity** 基础（2026年底开始）
2. 刷 **Ethernaut** + **Damn Vulnerable DeFi**（智能合约漏洞靶场）
3. 参加 **Code4rena / Sherlock** 审计竞赛（公开赛，赚钱 + 建声誉）
4. 目标公司：Trail of Bits、Halborn、Certik、ConsenSys Diligence

---

## 关键原则

1. **先蓝后红** — 理解防御才能更好攻击，不要跳过SOC思维
2. **OSCP是护照** — 没有它，顶级红队岗基本不看你简历
3. **CTF > 证书背诵** — 项目实战经历比一堆中级证书值钱
4. **Web3是差异化** — 99%的安全人不懂DeFi，懂就是竞争壁垒
